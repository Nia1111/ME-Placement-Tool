# 色彩の鍵

<ItemImage id="meplacementtool:key_of_spectrum" scale="4" />

**色彩の鍵**は、MEケーブル設置ツールの極めて重要なアップグレードです。これを装着することで、ツールは光と色を自在に操る力を得ます。

## 機能

色彩の鍵を**MEケーブル設置ツール**のアップグレードスロットに装着することで、以下の機能が解放されます:

- **色選択の解放**: ツールの設定画面から、17種類のAE2ケーブルの色を自由に選択できるようになります
- **自動染色**: ケーブル設置時、染料を消費することなく選択した色に自動で着色されます。これにより、色分けされたネットワークの構築が非常にスムーズになります。

## 入手方法

色彩の鍵は、特定のレシピを使用してクラフト可能です。
