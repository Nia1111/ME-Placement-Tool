# スペクトラムの鍵

<ItemImage id="meplacementtool:key_of_spectrum" scale="4" />

**スペクトラムの鍵**は、MEケーブル設置ツールの極めて重要なアップグレードパーツです。これを装着することで、ツールは光と色を自在に操る力を得ます。

## 機能

スペクトラムの鍵を**MEケーブル設置ツール**のアップグレードスロットに挿入することで、以下の機能が解放されます:

- **色選択の解放**: ツールの設定画面から、17種類のAE2ケーブルの色を自由に選択できるようになります
- **自動染色**: ケーブル設置時、追加の染料を消費することなく、選択した色に自動で変換されます。これにより、色分けされたネットワークの構築が非常にスムーズになります

## 入手方法

スペクトラムの鍵は、特定のレシピを使用してクラフトされます。
